> git 默认不会走代理  push代码有时候 会直接失败  你这种问题80%都是代理没有设置好，你开启了梯子，但是git工具没有设置相关的代理，导致git其实并没有走代理，一般输入以下命令设置代理即可


git config --global https.proxy 127.0.0.1:7897

git config --global http.proxy 127.0.0.1:7897
